# mscsia# Student Lifestyle Analysis

## Overview
This project analyzes how lifestyle habits such as study hours, sleep duration, social media usage, and physical activity influence students' academic performance (CGPA) and stress levels.

## Dataset
The dataset contains 100,000 observations describing student lifestyle patterns, stress levels, and academic outcomes.

## Methods
The analysis includes:
- Data exploration using pandas
- Data visualization using seaborn and matplotlib
- Linear regression modelling to examine factors affecting CGPA

## Tools
Python libraries used:
- pandas
- seaborn
- matplotlib
- statsmodels
